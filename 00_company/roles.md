# Roles

ใช้ไฟล์นี้กำหนดหน้าที่ของคนหรือระบบในทีม

| Role | หน้าที่ | เจ้าของ |
| :--- | :--- | :--- |
| Owner | ตัดสินใจภาพรวม อนุมัติมาตรฐาน และดูทิศทางช่อง | User Lika |
| Streamer | ไลฟ์จริง เตรียมเนื้อหา และสื่อสารกับคนดู | User Lika |
| Content Planner | วางแผนหัวข้อ ตาราง และ campaign | |
| Designer | ทำปก ภาพตาราง overlay และ asset | |
| Editor | ตัดคลิป highlight และ short video | |
| Moderator | ดูแล chat, community rules และกิจกรรมคนดู | |
| Automation | ดูแล script, prompt, template และ data workflow | |

## กฎการส่งต่องาน

- งานทุกชิ้นควรมีเจ้าของ
- ถ้างานยังไม่พร้อมใช้ ให้ใส่สถานะ Draft
- ถ้างานพร้อมโพสต์หรือใช้งาน ให้ใส่สถานะ Final

## สถานะปัจจุบัน

ตอนนี้ยังทำคนเดียวเป็นหลัก ดังนั้น User Lika รับบท Owner + Streamer ก่อน ส่วน role อื่นเป็นโครงสำหรับอนาคต
