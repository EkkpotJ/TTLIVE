# Streaming Availability

ไฟล์นี้บันทึกเวลาที่เหมาะสมสำหรับการสตรีม โดยอิงจากตารางชีวิตจริง

## Work Schedule

| วัน | เวลางานประจำ | หมายเหตุ |
| :---: | :---: | :--- |
| จันทร์-ศุกร์ | 09:00-18:00 | สตรีมได้ก่อนงานหรือหลังเลิกงาน |
| เสาร์-อาทิตย์ | ว่างมากกว่า | เหมาะกับไลฟ์ยาวหรือเกมที่ต้องใช้เวลา |
| วันหยุดนักขัตฤกษ์ | ว่างมากกว่า | เหมาะกับ event หรือไลฟ์พิเศษ |

## Primary Streaming Windows

| ช่วง | เวลา | เหมาะกับ |
| :--- | :---: | :--- |
| เช้า | 07:00-09:00 | Golden Spatula, เกมเบา, daily routine |
| เย็น/กลางคืน | 19:00 เป็นต้นไป | Wild Rift, Naraka, community live |
| เสาร์-อาทิตย์ | ยืดหยุ่น | ไลฟ์ยาว 2-4 ชั่วโมง |

## Current Capacity

- เป้าหมายเวลาสตรีมปัจจุบันประมาณ 15 ชั่วโมงต่อสัปดาห์
- แต่ละไลฟ์ควรมีขั้นต่ำ 1 ชั่วโมง
- ถ้างานประจำไม่ยุ่ง สามารถเพิ่มเป็น 2-4 ชั่วโมงต่อรอบได้

## Scheduling Rule

- วันทำงานควรเน้นรอบสั้นและชัด เช่น 07:00-09:00 หรือ 19:00-21:00
- เสาร์-อาทิตย์เหมาะกับรอบยาวหรือเกมที่ต้องใช้สมาธิ
- ถ้าเหนื่อย ให้ลดจำนวนรอบก่อนลดคุณภาพการไลฟ์
