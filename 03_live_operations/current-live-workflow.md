# Current Live Workflow

บันทึก workflow จริงของช่องโคโม๊ะ ณ ช่วงเริ่มต้น

## Before Live

- เปิด TikTok Live Studio
- ตั้งค่าไมค์
- ตั้งค่าเพลง
- ตั้งค่าจอหรือ capture เกม
- เตรียมเกมที่จะเล่น
- ตรวจการเชื่อมต่ออินเทอร์เน็ต
- ตรวจสภาพแวดล้อมรอบตัว
- เตรียมชื่อไลฟ์
- ยังไม่เปิดกล้องสตรีม

## During Live

- พูดคุยกับคนดู
- ทักทายคนที่เข้ามา
- ขอบคุณคนดู
- ถามคำถามคนดู
- อาจจดชื่อคนดูที่มาบ่อย
- อาจหาข้อมูลเกมหรือเพลงระหว่างไลฟ์
- เปิดเพลงตามสไตล์ช่อง

สิ่งที่ควรเริ่มจดแบบเบาๆ:

- ชื่อคนดูที่มาบ่อย
- คำถามที่คนดูถามซ้ำ
- ช่วงที่คนดูคุยเยอะ
- ช่วงที่ควรตัดเป็นคลิป
- ปัญหาเสียง/เน็ต/ภาพ

## After Live

ปัจจุบัน:

- พูดขอบคุณก่อนจบไลฟ์
- เปิดเพลงหลังจบ
- อยากมีหน้าจอจบพร้อมเพลง 1 เพลงก่อนปิดไลฟ์

สิ่งที่ควรเพิ่ม:

- กรอก live report แบบสั้น
- จด clip candidate อย่างน้อย 1 ช่วง
- จดปัญหาที่เกิดขึ้น
- จดสิ่งที่ควรปรับครั้งหน้า

## Common Problems

- ไลฟ์กระตุก
- ยังดึงดูดคนดูได้ไม่มาก
- ช่องยังผู้ติดตามน้อย
- การมองเห็นยังไม่เยอะ

## Suggested Fixes

ก่อน live:

- ตรวจ speed/connection แบบเร็ว
- ปิดโปรแกรมที่ใช้เน็ตหรือ CPU หนัก
- ตรวจ bitrate/resolution ใน TikTok Live Studio
- เตรียมชื่อ live ที่บอกเกมและกิจกรรมชัด
- เตรียม starting soon screen เพื่อให้เริ่มไลฟ์ดูจริงจังขึ้น

ระหว่าง live:

- ทักคนดูใหม่เร็วขึ้น
- ถามคำถามง่ายๆ เช่น เล่นเกมนี้ไหม / อยู่แรงค์ไหน / อยากให้ลองอะไร
- ถ้าคนดูเงียบ ให้พูดสิ่งที่กำลังคิดในเกมแทน
- mark ช่วงที่ดีสำหรับตัดคลิป

หลัง live:

- ใช้ offline/recap screen พร้อมเพลง 1 เพลง
- ขอบคุณคนดู
- บอกว่าจะเจอกันครั้งหน้าเมื่อไรถ้ารู้
- กรอก report สั้นภายในวันเดียวกัน
