# Channel Scene Plan

แผนภาพและ scene ที่เหมาะกับช่องโคโม๊ะในช่วงเริ่มต้น

## Current Assets

| รายการ | สถานะ |
| :--- | :--- |
| Logo | ใช้รูปตัวเอง |
| สีหลัก | Draft |
| ฟอนต์ | Draft |
| Background | ใช้ภาพจากเว็บหลักเกม |
| Mascot | ยังไม่มี |
| Game icons | ใช้จากภาพ/asset ที่ดาวน์โหลด |

## Scene Priority

### P0: ควรทำก่อน

- Starting soon screen
- Offline / ending screen
- Weekly schedule image

### P1: ทำหลังจากเริ่มนิ่ง

- In-live overlay
- Clip cover
- Recap post

### P2: ทำภายหลัง

- Mascot
- Full overlay pack
- Animated scene
- Sponsor overlay

## Starting Soon Direction

ควรเรียบ อ่านง่าย และบอก 3 อย่าง:

- วันนี้เล่นเกมอะไร
- เริ่มกี่โมง
- ช่องชื่อโคโม๊ะ / TikTok `@komonokimani`

## Ending Screen Direction

ใช้ก่อนปิดไลฟ์พร้อมเพลง 1 เพลง

ข้อความหลัก:

```text
ขอบคุณที่มาดูไลฟ์
เจอกันครั้งหน้า
```

ข้อมูลเสริม:

- เกมที่เล่นวันนี้
- TikTok handle
- ตารางไลฟ์ถัดไปถ้ารู้

## Overlay Direction

เพราะยังไม่เปิดกล้องและเล่นเกมมือถือ/PC หลายเกม overlay ควรเบามาก:

- logo เล็ก
- handle เล็ก
- event tag เช่น `เล่นกับคนดู`, `มือใหม่`, `ไต่แรงค์`
- ห้ามบัง UI เกม
